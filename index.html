About Pajabat

PAJABAT software was created by integrating the Kozeny-Carman's equation with 3 dimensional porous media sample to obtain it physical property values. This Software can be an alternative method for measuring permeability, porosity, tortuosity, and specific surface area of a rock sample that is simpler than conventional methods because this method measures rock physical properties without using any fluid injection. This software can also help the interpretation of a rock sample because it can display 2D layers and 3D models of the sampleâ€™s physical properties.

Method

This software is designed to be able to do numerical calculations of the physical value of a rock automatically with an easy-to-use UI that makes it easier for users to operate it. The work steps of this software are shown in figure.

  1. Tortuosity Measurement

    2. Tortuosity measurement works by calculating the connected path between one end of the axis to the other then divided by distance.

  2. Porosity Measurement

    3. Porosity can be calculated by comparing the pore area volume with the total rock valume.

  3. Specific Surface Area Measurement

    4. To calculate a specific path of the porous media which is still in the form of omages must be converted into binary form, then we calculate the specific surface area of the sample.

  4. Average Pore Measurement

    5. The calculation of average pore is done by using image processing functios that are quite commonly used by researchers so that they are easy to implement.

  5. Permeability Measurement

    6. Permeability measurement can be done by using Kozeny-Carman equation.
